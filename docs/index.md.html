Some of the things it can do: * Run on a computer * Exit with an exit code * Be a replacement for our old system, `theoldthing.exe` ## Invoking it This is a note by the way! From the root folder, run "thing" with the target name, e.g.: `thing Yes` * On Mac, Linux or Windows PowerShell, you'll want to use `./thing` instead of just `thing`. * Target and option names are case sensitive! * Several things to do can be specified: `thing Yes No FileNotFound` Additional options can be passed via `-sNAME=VALUE` arguments. Most common ones are CONFIG (used to indicate debug or release thing) and PLATFORM (used to indicate 32 vs 64 bit thing). ### Subsection on stuff Thing Name | Description |Notes ------------------------|--------------------|----- Yes | Yup! | No | Nope :( | FileNotFound | Doesn't find files | Pass `-sFIND_FILE=maybe` to maybe find them ## Markdeep syntax test Regular styling like **bold**, _italics_, ~~strikethrough~~, `inline code`, the stuff. * A * Bullet * List And: 1.
